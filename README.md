
# Certifyme.io

Certifyme.io is a Next.js-based application designed to streamline the process of adding names to certificates. It provides a user-friendly interface and a range of features for efficient certificate generation.

## Features

1. **Upload PDF Template:**
   - Select a PDF file of the certificate template from your local machine.
   - Supported file formats: PDF.

2. **CSV Integration:**
   - Utilize a CSV file to input a list of names for the certificates.
   - Ensure the CSV file follows the specified format.

3. **Manual Entry:**
   - Manually input names using a convenient form.
   - Validate input to prevent errors.

4. **Sample Certificate Generation:**
   - Generate a sample certificate to preview the output with a chosen name.

5. **Batch Certificate Generation:**
   - Generate certificates for multiple individuals with a single click.

6. **Download All Certificates:**
   - Download all generated certificates as a zip file.
   - Organize certificates in a structured folder hierarchy.

7. **Customizable Strips:**
   - Choose from provided strips.
   - Customize the position on the Y-axis.
   - Adjust offsets for precise placement.

8. **Preview Certificates:**
   - Get a preview of all the generated certificates before downloading.
   - Review and validate placement and formatting.
